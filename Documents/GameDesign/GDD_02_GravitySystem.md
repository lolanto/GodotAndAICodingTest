# 第2章：重力系统

## 2.1 全局重力操控

### 2.1.1 方向切换

玩家可将**全局重力**在以下预设方向中切换：

```
↑ 上 / ↓ 下（默认）/ ← 左 / → 右
以及四个斜角方向（需解锁）：↗ ↘ ↙ ↖
```

- **切换延迟**：重力方向切换后，有约 **0.3 秒**的过渡缓冲期，防止操控感突兀。方向值**瞬间生效**，"缓冲感"来自物体原有速度的惯性保留，而非方向向量本身的插值。
- **惯性保留**：切换时，场景内所有物体保留切换前的速度分量，随后在新重力方向下继续加速。这是弹射类谜题的核心利用点。
- **视觉反馈**：切换时屏幕边缘出现方向光晕，场景内所有动态物体产生短暂的拖影特效。

### 2.1.2 全局重力的作用范围

全局重力**作用于**：

- 玩家角色
- 无固定锚点的动态物体（木箱、球体、液体等）
- 敌人（除非敌人有自主飞行/固定AI）
- 可触发机关的可动部件

全局重力**不作用于**：

- 固定背景建筑
- 已被"锚定"状态的物体
- 粒子特效（仅视觉层）

## 2.2 局部重力场

局部重力场（Local Gravity Field，简称 **LGF**）是玩家可以在特定区域内放置、覆盖全局重力规则的"重力泡泡"。

### 2.2.1 LGF 属性

| 属性 | 说明 | 限制 |
|------|------|------|
| 形状 | 圆形（基础）/ 矩形（解锁）/ 扇形（高级解锁） | — |
| 半径/范围 | 可调节，消耗更多 GE | 最大半径由等级决定 |
| 方向 | 独立于全局重力，可自由设定 | — |
| 强度 | 独立于全局重力强度 | 0× ~ 5×（局部场可超过全局上限） |
| 持续时间 | 分两类：①玩家技能 LGF 消耗 GE 维持，放置后持续扣能；②场景机制 LGF 由关卡逻辑控制（开关、计时器等），不受 GE 约束 | 主动或能量耗尽时消失（玩家技能）；由关卡事件决定（场景机制） |
| 叠加数量 | **玩家技能 LGF**：同时最多 **3** 个（默认），由技能 CD 与存续时间约束，可通过道具升级上限至 **6** 个。**场景机制 LGF**：无数量限制，由关卡设计决定 | — |

> **两种来源的区别**：
> - **场景机制 LGF**：由关卡设计师预置于场景中，通过游戏机关（压力板、开关、计时器等）触发激活或停用，其状态变化会被时间回溯系统记录和还原。
> - **玩家技能 LGF**：玩家在运行时主动释放，消耗 GE 维持，默认同屏最多 3 个（升级后最多 6 个），到期或手动移除后消失，同样参与时间回溯。槽位在代码层预分配至上限（6 个），升级时解锁对应槽位，不动态增删节点。

### 2.2.2 LGF 与全局重力的叠加规则

当物体处于 LGF 内时：

```
实际重力向量 = LGF重力向量 × (1 - 混合因子) + 全局重力向量 × 混合因子
```

- **完全覆盖模式**（混合因子 = 0）：LGF 完全替代全局重力。
- **叠加模式**（0 < 混合因子 < 1）：两种重力部分融合，产生斜向合力。
- 默认为**完全覆盖模式**，叠加模式需特殊道具或关卡授权。

### 2.2.3 多个 LGF 叠加

当多个 LGF 区域重叠时，取**最后放置的 LGF**优先（后置优先）。
未来扩展可考虑"重力合力模式"（取向量和），作为高难度谜题元素。

## 2.3 重力强度

重力强度以**标准重力 g₀** 为基准单位，支持连续调节：

| 强度区间 | 效果描述 | 典型用途 |
|----------|----------|----------|
| 0× (零重力) | 物体漂浮，保留惯性 | 漂浮谜题、零重力区域 |
| 0.1× ~ 0.5× | 极轻重力，跳跃高度大幅增加 | 高跳越障、减速降落 |
| 0.5× ~ 1× | 低重力，轻盈感 | 弹跳平台区域 |
| **1×（默认）** | 标准重力 | 基准状态 |
| 1× ~ 2× | 高重力，角色移动变重 | 压力谜题，限制敌人 |
| 2× ~ 3× | 极高重力，大部分跳跃受阻 | 碾压机关、弹射加速 |

- **强度调节方式**：拨盘（手柄模拟摇杆 / 键盘数字键快捷切换）。
- **变化速率**：强度变化不是瞬时的，有 **0.2 秒**的插值过渡，避免物理突变导致穿模。

## 2.4 重力与物体的交互

### 2.4.1 物体类型分级

| 类型 | 全局重力 | LGF | 强度缩放 | 备注 |
|------|---------|-----|----------|------|
| 标准物体 | ✅ | ✅ | ✅ | 木箱、石块等 |
| 轻质物体 | ✅（加倍位移） | ✅ | ✅ | 羽毛、纸张，视觉夸张效果 |
| 重质物体 | ✅（减半位移） | ✅ | ✅ | 铁球，碾压类机关 |
| 液体 | ✅（特殊流体模拟） | ✅（局部翻转） | 有限 | 可倒灌机关 |
| 玩家 | ✅ | ✅ | ✅ | 有跑步/攀爬补偿 |
| 敌人（地面型） | ✅ | ✅ | ✅ | 可被反重力击倒 |
| 敌人（飞行型） | ❌ | 可选 | ❌ | 部分区域有例外 |
| 投射物 | ✅ | ✅ | ✅ | 子弹弧度随重力变化 |
