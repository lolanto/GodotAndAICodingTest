# 第4章：重力 × 时间 联动机制

这是本游戏最核心的设计亮点。以下描述两者相互影响的具体规则与设计范式。

---

## 4.1 重力影响时间

### 4.1.1 引力时间膨胀（Gravitational Time Dilation）

> 灵感来源于广义相对论：强引力场中时间流速变慢。

**规则**：

- 当**全局重力强度 > 1×** 时，场景内时间流速随强度**连续降低**（无离散触发阈值，效果由公式决定）。
- 计算公式（简化）：

$$\tau = \tau_0 \times \left(1 - k \cdot \frac{g - g_0}{g_{max}}\right)$$

其中 $\tau$ 为当前时间流速，$\tau_0 = 1.0$，$k = 0.4$，$g_{max} = 3g_0$。

- **g = 1×** → $\tau = 1.0$（正常）
- **g = 2×** → $\tau \approx 0.87$（轻微减速）
- **g = 3×** → $\tau \approx 0.73$（明显减速）

**玩法用途**：

- 玩家可以在不直接使用慢动作技能的情况下，通过拉高重力**被动减慢时间**，节省 CE。
- 高重力 + 被动慢时间：让玩家能在敌人的慢动作状态下精准完成操作。
- 谜题设计：某个机关只能在"时间流速低于 0.8×"时被触发，玩家需要通过提升重力来满足条件。

### 4.1.2 零重力时间失锚（Temporal Unanchoring in Zero-G）

**规则**：

- 当全局重力设为 **0×（零重力）** 持续 **3 秒**后，CE 回复速率**提升 3 倍**。
- **前提**：该加成仅在 CE **没有被消耗**时生效。若玩家同时触发了时间技能（回溯 / 慢动作等），消耗优先，回复（含加成）暂停。

**叙事解释**：在零重力下，时空结构变得松弛，时间能量更容易积蓄。

**玩法用途**：

- 玩家可以在安全区域维持零重力状态快速恢复 CE，为后续高强度时间操控做准备。
- 区域设计：某区域需要连续多次回溯（CE 消耗大），玩家可先在零重力区充分积蓄 CE 再进入。

## 4.2 时间影响重力

### 4.2.1 时间回溯复原重力状态

- 时间回溯时，**重力方向、强度、所有 LGF** 完全同步回溯。
- 这意味着：玩家误操作导致的重力混乱可以被完全撤销。

**深度玩法**：

- 利用"重力状态回溯"创造"重力因果循环"谜题：
  - 例：玩家在 T=5s 时放置 LGF 使铁球落入凹槽（触发机关A），
    在 T=8s 时触发了另一机关B，
    此时回溯到 T=5s 但**保留机关B已触发的状态**（通过局部回溯），
    使铁球在新的轨迹下同时触发机关A+B。

### 4.2.2 慢动作中的重力精细控制

- 在**慢动作状态**下，重力操控的**响应精度提升**：
  - LGF 放置的位置判定从格子对齐变为**像素级对齐**。
  - 重力强度拨盘步进从 0.25× 变为 **0.05×**（细粒度调节）。
- 玩法：某些谜题要求重力强度精确到 0.75×（半格子调节无法达到），必须用慢动作操控。

### 4.2.3 时间加速放大重力效果

- 在**时间加速（2×）状态**下：
  - 物体的重力加速效果等效于当前重力强度的 **2 倍**（时间流速为 2×，相同真实时间内速度累积量 $v = g \cdot 2t$ 翻倍，与时间倍率一致）。
  - 玩法：利用时间加速将轻质物体快速加速到高速度，随后切回正常时间利用其动能。

### 4.2.4 时间冻结中的"静态重力雕塑"

- 使用特殊道具**时间定格锚**（Chrono Anchor），可以在特定物体上冻结时间。
- 被冻结的物体**不受任何重力影响**（LGF 也无法移动它）。
- 玩法：将冻结物体作为平台、盾牌、支撑点，搭配重力变化创造复杂路径。

## 4.3 联动谜题范式

以下是几类利用重力×时间联动设计的具体谜题模板：

### 范式 A：「时光弹弓」（Chrono Slingshot）

```
步骤：
1. 将重物放置在高处（借助反重力）
2. 时间回溯，让重物恢复到原始低位
3. 在回溯结束瞬间，切换重力方向
4. 重物因惯性向新方向高速弹射，触发远处机关
```

**核心**：利用回溯+惯性保留，制造"无中生有"的弹射力量。

---

### 范式 B：「重力时间沙漏」（Gravity Hourglass）

```
步骤：
1. 场景中有沙漏型容器，沙粒需从上半部流向下半部触发机关
2. 玩家翻转全局重力，沙粒开始向新方向运动
3. 激活慢动作（全局减速至 0.5×），在沙粒通过精准通道入口时放置 LCF（局部慢时间场）
4. 解除慢动作后，沙粒在 LCF 的极慢时间中缓缓流过精准通道
5. 到达目标位置后，解除 LCF，沙粒恢复正常速度完成填充
```

**核心**：慢动作提供精确放置 LCF 的操作窗口 + LCF 的持续时间控制，组合解决流体谜题。

---

### 范式 C：「因果轮回」（Causal Loop）

```
步骤：
1. 玩家A时刻操作某机关（如按下按钮），触发B时刻的门开启
2. 玩家通过门，发现需要在B时刻放置LGF才能让B时刻的门保持开启
3. 玩家回溯到A时刻前，先放置LGF，再按按钮
4. 正向时间线中，LGF存在 → 门长时间开启 → 玩家顺利通过
```

**核心**：时间回溯与空间物体（LGF）状态组合，形成可解的因果悖论谜题。

---

### 范式 D：「重力驯服」（Gravity Taming）—— 战斗应用

```
敌人：重甲骑士（受高重力影响移动极慢）
步骤：
1. 提高全局重力到 3×（触发时间膨胀，时间流速降至0.73×）
2. 重甲骑士几乎无法移动
3. 玩家利用低速时间精准攻击弱点
4. 若操作失误，激活慢动作（全局减速至 0.5×，玩家与骑士同步放慢）
5. 在慢动作下精准找准弱点完成攻击
```

**核心**：重力强度 → 被动时间减速 → 敌人减速 → 精准战斗窗口；慢动作作为应急手段（玩家同样减速，属高操作要求）。

---

### 范式 E：「逆流而上」（Against the Current）

```
场景：有一条向右高速流动的液体通道，液体中有敌人
步骤：
1. 玩家使用时间回溯，将液体流向倒转（液体向左流）
2. 同时在通道左侧放置 LGF（向上），液体流到左端被向上吹出
3. 被向上吹出的液体淹没上方的电气机关，导致短路开门
4. 玩家切换回正常时间，通过门
```

**核心**：时间回溯改变液体运动方向 + LGF 改变液体终点，间接触发机关。
