# 第1章：游戏概述

## 1.1 核心世界观：箱子（The Box）

整个游戏世界是一个密闭的**巨型箱子**（The Box）。玩家从箱子内部某处醒来，**最终目标是找到并离开这个箱子**。

箱子并非单一房间，而是由数十个相互毗邻的**区域（Zone）**构成的复杂空间。区域在几何上彼此相邻，但初始状态下大多数区域间存在**隔离屏障**，玩家必须通过解谜来打通屏障、开辟通路。

游戏**没有传统的关卡切换界面**。玩家通过**改变重力方向在箱子中自由移动**——将重力指向右边，角色就向右"坠落"；将重力反转，角色就能落向天花板。箱子的几何结构本身即是谜题的一部分：有些区域只有在重力朝向特定方向时才可进入。

## 1.2 核心循环

```
探索箱子 → 发现隔离屏障 / 不可达区域 → 解开谜题
                                              ↓
                              解锁新区域 + 获得新能力
                                              ↓
              携带新能力回溯旧区域，到达之前不可达的位置
                                              ↓
                      不可达位置中隐藏解开新区域谜题的关键
                                              ↓
                              继续解谜 → 逼近出口
```

这个循环是**非线性**的：

- 玩家在**区域 A** 中，某些角落因缺乏特定能力而无法到达——这不妨碍向前推进。
- 玩家来到**区域 B**，获得新能力后，意识到可以返回区域 A 的某个角落。
- 区域 A 不可达的位置往往隐藏着**辅助解开区域 B 谜题的关键道具或线索**。
- 这形成了区域之间天然的**双向依存关系**，鼓励探索与回溯，而非线性推进。

## 1.3 设计愿景

- **重力**既是玩家在箱子中穿行的"移动引擎"，也是改造世界的"空间画笔"。
- **时间**是玩家容错与策略的"橡皮擦"，允许修正错误、构建因果链。
- **箱子结构**是叙事与玩法的统一载体——每解开一道谜题，都是在揭开箱子的秘密、向出口迈近一步。
- 两者的**深度耦合**产生"空间-时间谜题"，使每个区域都存在多种探索路径与解法。

## 1.4 操控哲学

| 维度 | 操控方式 | 代价 |
|------|----------|------|
| 重力方向 | 主动切换（4/8 方向 或 自由角度） | 消耗重力能量（Gravity Energy, GE） |
| 重力强度 | 拨盘式调节（0× ~ 3×标准重力） | 消耗 GE，强度越高耗能越快 |
| 时间回溯 | 按住触发键，倒回最多 N 秒 | 消耗时间能量（Chrono Energy, CE） |
| 时间流速 | 慢动作 / 加速 | 消耗 CE，影响全局或局部 |
