# 第6章：数值框架参考

## 6.1 能量系统基准

| 参数 | 默认值 | 最大升级值 |
|------|--------|-----------|
| GE 上限 | 100 | 200 |
| GE 自然回复 | 8/秒 | 15/秒 |
| CE 上限 | 100 | 200 |
| CE 自然回复 | 5/秒 | 12/秒 |
| 时间回溯最大时长 | 5 秒 | 10 秒 |
| 同屏 LGF 数量 | 3 | 6 |
| 同屏 LCF 数量 | 1 | 3 |

> **消耗与恢复互斥原则**：GE 和 CE 的消耗与自然回复在同一帧内只执行其一。
> - **GE**：有活跃玩家技能 LGF 消耗 GE 时，自然回复暂停；无消耗时才回复。
> - **CE**：处于任意时间操控状态（回溯 / 慢动作 / 加速 / LCF 维持）时，自然回复暂停；无消耗时才回复（含零重力 3× 加成）。

## 6.2 重力切换冷却

| 操作 | 冷却时间 |
|------|----------|
| 全局重力方向切换 | 1.5 秒 |
| 全局重力强度大幅变化（>1×阶跃） | 0.5 秒 |
| LGF 放置/移除 | 0.3 秒/个 |

## 6.3 时间联动触发阈值

| 联动效果 | 触发条件 |
|----------|----------|
| 引力时间膨胀开始生效 | g > 1×（连续生效，无离散阈值） |
| 引力时间膨胀最大效果 | g = 3× → τ = 0.73 |
| 零重力时间失锚 | g = 0× 持续 ≥ 3 秒 |
